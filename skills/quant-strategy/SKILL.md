## 铁律（所有流程前置）

> **每次选股/操作前必须朗读铁律**

1. **止损-3%无条件执行**
2. **禁止高开追买（涨幅>3%不买）**
3. 同时持仓≤3只，单票仓位≤30%
4. 最佳买入时机：尾盘14:30后（不提前买）
5. 昨夜纳指>2% → 优先半导体/芯片/新能源

**情绪判定为"恐慌杀跌"或"偏弱退潮"时，不推荐任何买入标的，仅提示空仓观望。宁可错过不可做错。**

**板块涨停潮内所有标的整体排除，等次日；高开>6%立即放弃。**

## 持仓分析流程

**前置检查（必执行）**：
0. 调用问财查询大盘环境：上证指数涨跌幅、涨跌家数比、北向资金、涨停家数、炸板率
1. 硬性暂停条件：上证 < -1.5% 或 下跌家数 > 上涨家数×2 → 停止所有买入，仅做持仓风控
2. 降仓条件：北向流出>50亿 / 涨停<20家 / 炸板率>50% → 总仓位按比例缩减

**正式流程**：
1. 读取持仓 memory（见下方持仓结构）
2. 调用问财技能拉取实时行情、基本面、资金数据
3. 计算浮盈浮亏、仓位匹配度、风险等级
4. **输出格式**：操作建议 + 数据依据 + 风险等级 + 适用周期

## 选股咨询流程

**前置检查（必执行）**：
大盘环境过滤检查，不通过则终止选股。

**正式流程**：
1. 解析用户选股需求
2. 调用问财条件筛选接口
3. 返回 3–5 只标的 + 筛选逻辑
4. 东方财富辅助验证（如换手率>30%排除）
5. 不承诺收益，不给出"必买"指令

**⚠️ 14:30 二次确认（必执行）**：
1. 重新查询候选标的最新涨幅
2. 淘汰涨幅突破6%上限的标的
3. 确认最终买入标的

## 情绪驱动选股流程

**触发条件**：用户盘中输入市场情绪判断（如"今天偏多，主线是新能源"）

**执行流程**：
0. 大盘环境过滤检查 → 硬性暂停则终止
1. 解析情绪输入 → 确定5维度评分 + 市场状态标签
2. 市场热点分析（finance-news东方财富板块排行 + 主线识别）
3. 读取 strategy-selector.json 策略映射
4. 执行各策略问财查询
5. 交叉验证：先取交集（多策略共振），不足则并集+加权打分
6. 东方财富辅助验证：换手率>30% → 排除
7. 输出 Top 5：名称 + 代码 + 策略来源 + 风险等级 + 仓位建议 + 适用周期
